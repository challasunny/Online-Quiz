# Online-Quiz Application
This is a simple web-based Online Quiz application developed using **HTML, CSS, and Express.js**.
## Features
-User login with credentials
-Quiz starts only if credentials are valid
-Displays some programming languages
-Based on the selected language you will get questions related to that language
-Simple and clean UI
## Technologies Used
-HTML
-CSS
-Express.js
## Project Stucture
-'public/' -> Frontend files(HTML, CSS)
-'routes/' -> Route handling
- 'views/' -> Templates(pug)
- 'server.js' -> Main backend server
## How to Run
1.Intall dependencies:
  npm install
2.Start the server:
  nodemon server.js
3.open browser:
  http://localhost:2999
